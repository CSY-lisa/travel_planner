### 🔧 一次性設定：clasp（以後每次改完 GAS 只要 `clasp push`）

clasp 是 Google 官方工具，讓你從終端機直接把本地 `gas/` 資料夾推送到 GAS，**不需要手動複製貼上**。

**安裝並登入：**
```bash
npm install -g @google/clasp
clasp login   # 會開瀏覽器授權，完成後回終端機
```

**填入 Script ID：**
1. 開啟 [Google Apps Script](https://script.google.com/) → 選你的 travel_planner 專案
2. 左側「⚙️ 專案設定」→ 複製「指令碼 ID」
3. 編輯 `gas/.clasp.json`，把 `填入你的_SCRIPT_ID` 換成實際 ID：
```json
{
  "scriptId": "你的實際Script ID",
  "rootDir": "."
}
```

**之後每次更新 GAS，只需要：**
```bash
cd "/Users/lisa/Library/CloudStorage/GoogleDrive-chycl06@gmail.com/我的雲端硬碟/AI_project/csk/travel_planner/gas"
clasp push
```
全部 4 個 `.gs` 檔案一次上傳完畢。

### 🟡 Python 批次編輯器（Step 5–8）

#### Step 5：取得 Service Account JSON
1. 前往 [Google Cloud Console](https://console.cloud.google.com/)
2. 「IAM 與管理」→「服務帳戶」→「建立服務帳戶」（或選已有的）
3. 「金鑰」→「新增金鑰」→ 類型選 **JSON** → 下載檔案
4. 記住 JSON 裡的 `client_email` 值

#### Step 6：更新 `.env`
```
# 把這行的路徑換成你實際下載的 JSON 檔案位置
GOOGLE_SERVICE_ACCOUNT_JSON=/Users/lisa/路徑/service_account.json
```

#### Step 7：Google Sheets 共用給 Service Account
1. 開啟旅遊計畫試算表
2. 右上角「共用」
3. 貼上 Step 5 的 `client_email`（格式：`xxx@xxx.iam.gserviceaccount.com`）
4. 設定為「**編輯者**」→ 送出

#### Step 8：驗證 Python 連線
```bash
cd "/Users/lisa/Library/CloudStorage/GoogleDrive-chycl06@gmail.com/我的雲端硬碟/AI_project/csk/travel_planner"
python3 -c "
import gspread, os
from google.oauth2.service_account import Credentials
from dotenv import load_dotenv
load_dotenv()
creds = Credentials.from_service_account_file(os.getenv('GOOGLE_SERVICE_ACCOUNT_JSON'), scopes=['https://www.googleapis.com/auth/spreadsheets'])
gc = gspread.Client(auth=creds)
ss = gc.open_by_key(os.getenv('SHEET_ID'))
print('Connected:', ss.title)
"
```
期待輸出：`Connected: [試算表名稱]`
